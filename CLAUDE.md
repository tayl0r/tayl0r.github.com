# Project: taylorsteil.com

Personal site built with Vite + Three.js + TypeScript.

## Commands

- `pnpm run test` — type-check and lint (tsc --noEmit && biome check)
- `pnpm run build` — full build (tsc && vite build)
- `pnpm run format` — auto-format with biome
- `pnpm run dev` — local dev server on port 4100

## Rules

- Always run `pnpm run test` after making changes and before every commit.
- Fix any type errors or lint issues before committing.
